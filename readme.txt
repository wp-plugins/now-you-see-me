Hide your admin panel ... like a boss!

== Description ==

This plugin was forked from another awesome plugin called [Zing!](https://wordpress.org/plugins/zing/)

This plugin is perfect for:

* Playing an [April Fools' Day](http://en.wikipedia.org/wiki/April_Fools%27_Day) joke
* A good chuckle
* Pissing off your colleagues
* Blowing off your work
* Freaking out your boss
* Increasing hacker bounce rate
* Getting yourself fired

Press the `h` key to toggle _Now You See Me_ on and off.

**Development of this plugin is done [on GitHub](https://github.com/jonorherrington/NowYouSeeMe). Pull requests welcome. Please see [issues reported](https://github.com/jonorherrington/NowYouSeeMe/issues) there before going to the plugin forum.**
